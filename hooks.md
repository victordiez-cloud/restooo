# Hooks d’état

## useState

    Permet d’ajouter un état local à un composant fonctionnel.
    https://react.dev/reference/react/useState

## useReducer

    Permet de gérer un état complexe avec une logique centralisée (pattern reducer).
    https://react.dev/reference/react/useReducer

# Hooks de contexte

## useContext

    Permet d'accéder aux données d’un Context React sans passer les props.
    https://react.dev/reference/react/useContext

# Hooks de cycle de vie / effets

## useEffect

    Permet d'exécuter du code après le rendu du composant (API, subscriptions …).
    https://react.dev/reference/react/useEffect

## useLayoutEffect

    Fonctionne comme useEffect, mais s’exécute avant.
    https://react.dev/reference/react/useLayoutEffect

## useInsertionEffect

    Permet d’injecter du CSS avant le rendu DOM.
    https://react.dev/reference/react/useInsertionEffect

# Hooks de référence

## useRef

    Permet de stocker une valeur persistante entre les rendus ou d'accéder directement à un élément DOM.
    https://react.dev/reference/react/useRef

# Hooks de performance

## useMemo

    Permet de mémoriser le résultat d’un calcul coûteux pour éviter de le recalculer à chaque render.
    https://react.dev/reference/react/useMemo

## useCallback

    Permet de mémoriser une fonction pour éviter qu'elle soit recréée à chaque rendu.
    https://react.dev/reference/react/useCallback

# Hooks d’ID

## useId

    Génère un identifiant unique stable utile pour les formulaires et l’accessibilité.
    https://react.dev/reference/react/useId

# Hooks de concurrence

## useTransition

    Permet de marquer certaines mises à jour comme non urgentes pour améliorer la fluidité de l’interface.
    https://react.dev/reference/react/useTransition

## useDeferredValue

    Permet de retarder la mise à jour d’une valeur afin d’éviter les ralentissements de l’interface.
    https://react.dev/reference/react/useDeferredValue

# Hooks liés aux formulaires

## useFormStatus

    Permet d'obtenir l’état d’un formulaire en cours de soumission.
    https://react.dev/reference/react-dom/hooks/useFormStatus

## useFormState

    Permet de gérer l’état d’un formulaire côté serveur.
    https://react.dev/reference/react-dom/hooks/useFormState

# Hooks serveur

## use

    Permet d'utiliser des promesses ou des ressources directement dans un composant.
    https://react.dev/reference/react/use
